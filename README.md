\# Atividade Prática – Teoria dos Conjuntos em Python



\## Descrição

Este projeto foi desenvolvido como atividade prática da disciplina \*\*Lógica e Matemática Discreta\*\*

do curso de \*\*Engenharia da Computação – UFMA\*\*.



O objetivo é aplicar os principais conceitos da \*\*Teoria dos Conjuntos\*\*, como união, interseção,

diferença, complemento e cardinalidade, utilizando a linguagem Python.



\## Objetivo do Programa

O programa trabalha com dois conjuntos:

\- \*\*Conjunto A\*\*: definido pelo usuário (entre 4 e 8 elementos).

\- \*\*Conjunto B\*\*: gerado aleatoriamente pelo programa (entre 4 e 8 elementos).



A partir desses conjuntos, são realizadas operações fundamentais da teoria dos conjuntos.



\## Tipo de Dados Utilizado

Os elementos dos conjuntos são \*\*strings\*\*, o que permite flexibilidade na entrada e facilita

a visualização dos resultados.



\## Como Executar

1\. Certifique-se de ter o Python 3 instalado.

2\. Clone este repositório.

3\. Execute o arquivo principal:



```bash

python conjuntos.py
